# Brand Visual System

Version 1.0

---

# Core Brand Symbol

The brand is built around a geometric sharpener.

Not a realistic sharpener.

Not a product illustration.

A symbolic sharpener.

The sharpener represents:

- Refinement
- Intelligence
- Learning
- Improvement
- Craftsmanship
- Creation
- Transformation

The symbol communicates:

"Raw ideas enter. Better ideas emerge."

This is the foundation of the entire visual identity.

---

The visual system should feel:

---

# Symbol Design Language

Every symbol in the ecosystem should feel like it belongs to the same family as the sharpener logo.

The sharpener logo is the master reference.

All future symbols should inherit its geometry.

---

# Shape Principles

## Geometric Construction

All symbols must be constructed from:

- polygons
- trapezoids
- angled planes
- cut corners
- faceted surfaces

Avoid:

- organic curves
- soft blobs
- rounded cartoon forms

The geometry should feel engineered.

---

## Monolithic Forms

Every icon should feel carved from one object.

Never assemble icons from many floating parts.

Good:

Single unified structure.

Bad:

Collections of disconnected shapes.

---

## Faceted Surfaces

Objects should feel:

- sculpted
- cut
- carved
- manufactured

Like a precision tool.

Every edge should feel intentional.

---

# Negative Space Language

Negative space is a primary design element.

The sharpener uses:

- blade slot
- screw hole
- internal channels

Future symbols should use the same philosophy.

Meaning is created through subtraction.

Not decoration.

---

# Visual Weight

All symbols should have:

- thick mass
- strong silhouettes
- substantial presence

Avoid:

- thin line icons
- delicate details
- outline-only graphics

The logo should feel durable.

---

# Edge Treatment

Edges should be:

- sharp
- clean
- geometric

Never:

- bubbly
- cartoonish
- overly rounded

Minor bevels are acceptable.

Soft corners are not.

---

# Symbol Color

Never use pure white.

Preferred symbol color:

#F5F0E6

Alternative:

#EEE7D8

The symbol should feel warm.

Not sterile.

---

# Material Style

The logo should feel like:

- matte ceramic
- carved paper
- sculpted clay
- precision-cut material

Not:

- chrome
- glass
- metal
- glossy plastic

---

# Lighting Style

Soft studio lighting.

Characteristics:

- subtle shadows
- gentle highlights
- low contrast depth

Avoid:

- hard reflections
- dramatic lighting
- realism

The logo is symbolic.

Not a product render.

---

# Composition Rules

The symbol should occupy:

60%–75% of the canvas.

Centered.

Floating.

Surrounded by generous negative space.

The logo should never feel crowded.

---

# Secondary Graphic Elements

Use only elements derived from the logo.

Examples:

- blade-slot lines
- angular cuts
- faceted shapes
- geometric spark forms

Avoid generic startup graphics.

Every decorative element should feel like it could have been carved from the sharpener.

---

# Icon System

Future icons should follow the same visual language.

Examples:

Knowledge:
Faceted book glyph

Memory:
Carved archive block

Search:
Angular lens symbol

Ideas:
Geometric spark

Agent:
Precision-crafted intelligence glyph

Every icon should feel like it belongs to the same toolset.

---

# Motion Language

Movement should feel:

- precise
- mechanical
- intentional

Like a sharpening process.

Not playful bouncing.

Not elastic motion.

Think:

refine → shape → improve

---

# Brand Metaphor

The sharpener is not stationery.

The sharpener is a philosophy.

Input:
Raw thought.

Process:
Refinement.

Output:
Clarity.

The entire visual system should reinforce this idea.

Every screen.

Every icon.

Every illustration.

Every animation.

Everything should communicate:

"We sharpen ideas."
